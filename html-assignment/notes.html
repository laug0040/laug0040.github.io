5 HTML Tags I've learnt

  • The "p" tag is used for making paragraphs
  • The "ul" tag is used for making unodered lists
  • The "ol" tag is used for making ordered lists
  • The "img" tag is used to import images typically from the images folder
  • The "blockquote" tag is used to quote something that is from an external source
